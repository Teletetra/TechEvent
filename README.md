# TechEvent

A modern JavaScript-based event management and scheduling application designed to simplify tech event planning and coordination.

## About

**TechEvent** is a lightweight, web-based solution for managing technology events. Whether you're organizing a conference, hackathon, meetup, or webinar, TechEvent provides the tools to streamline event creation, scheduling, and attendee management.

## Features

- 📅 **Event Scheduling** - Create and manage events with ease
- 👥 **Attendee Management** - Track and organize event participants
- 📋 **Event Details** - Comprehensive event information and metadata
- 🎯 **User-Friendly Interface** - Intuitive design for seamless experience
- 📱 **Responsive Design** - Works across desktop and mobile devices

## Technology Stack

- **Frontend:** JavaScript (99.1%)
- **Markup:** HTML (0.9%)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- Modern web browser

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Teletetra/TechEvent.git
cd TechEvent
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser and navigate to `http://localhost:3000`

## Project Structure

```
TechEvent/
```

## Usage

1. **Create an Event** - Click "New Event" to create a new tech event
2. **Add Details** - Fill in event information (date, time, location, description)
3. **Manage Attendees** - Add and track event participants
4. **Share** - Get shareable links or export event details
